Ticket support

What to check before contacting support

Confirm the event details

Check the event page for date, start time, venue, entry requirements, age restrictions, and organizer updates.

Review your order

Open your dashboard and verify the ticket type, quantity, payment status, and QR ticket availability.

Save proof of the issue

If something looks incorrect, keep screenshots, receipt details, and any message from the organizer.

Contact support with context

Include the event name, order email, ticket quantity, and a clear description of the problem so the team can respond faster.
